# FederatedAKM
This repository contains supplementary material for de Vries, L. and de Boer, R.C.: "Towards Federated Architecture Knowledge Management: Practices for Scalable, Collaborative Modeling". In: 23rd IEEE International Conference on Software Architecture - Software Architecture in Practice (ICSA-SAiP 2026)

The files in this git-repository are ArchiMate repositories that can be opened with the open source tool Archi (https://www.archimatetool.com/). The repositories illustrate the federated approach - with domain repositories with domain-specific elements and core repositories with reusable elements - that is outlined in the paper.
